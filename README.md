# ⚔️ Sangre y Ceniza - E-commerce Vikingo

Esta es una tienda en línea temática desarrollada para el MVP del módulo 2 de Frontend. Se especializa en réplicas de grado museo de armamento y joyería nórdica.

## 🚀 Repositorio
Puedes encontrar el código fuente y el historial de desarrollo aquí:
(https://github.com/Xiihib/ecommerce-frontend-m2)

## 🛠️ Tecnologías Utilizadas
* **HTML5 Semántico**: Estructura clara y accesible.
* **Bootstrap 5**: Layout responsivo y componentes de UI.
* **JavaScript (ES6+)**: Gestión de carrito, manipulación del DOM y lógica de negocio.
* **LocalStorage**: Persistencia del "Botín de Guerra" entre sesiones.
* **CSS3 Personalizado**: Estilos temáticos medievales y animaciones.

## 📋 Requisitos del MVP Cumplidos
- [x] **Home**: Grilla de productos dinámica.
- [x] **Detalle**: Vista expandida con especificaciones técnicas.
- [x] **Carrito**: Gestión de cantidades y persistencia.
- [x] **Responsive**: Diseño adaptado para móviles (≤ 420px) y escritorio.
